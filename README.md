# 🤖 SmartBot - Telegram Bot

A smart and versatile Telegram bot built with Python, designed to assist users with various tasks including AI chat, advice, quotes, and more.

## ✨ Features

- 🤖 **AI Chat**: Intelligent conversation capabilities
- 💡 **Daily Advice**: Get random advice
- 📝 **Inspirational Quotes**: Motivational quotes
- ❓ **Q&A**: Ask questions and get answers
- ⚙️ **Settings**: Customizable user preferences
- 🔘 **Interactive Buttons**: Easy-to-use interface

## 🚀 Commands

| Command | Description |
|---------|-------------|
| `/start` | Welcome menu |
| `/help` | Show help |
| `/chat [text]` | Chat with AI |
| `/advice` | Get advice |
| `/quote` | Get quote |
| `/ask [question]` | Ask a question |
| `/settings` | Change settings |

## 🛠️ Technologies Used

- Python 3.9+
- python-telegram-bot library
- Requests library
- Free public APIs (no keys required)
